# ADR-005: MIRIX Multi-Agent System

## Status
Active

## Context
We need a complex memory handling system (episodic, semantic, etc.).

## Decision
**Reuse the MIRIX agent design and types.** Do not invent a new agent system from scratch.

## Consequences
- **Positive**: Proven model, faster implementation (reusing concepts).
- **Negative**: Need to port Python reference implementation to Rust (per ADR-009).
